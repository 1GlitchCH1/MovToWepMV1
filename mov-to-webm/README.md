# MOV → WebM конвертер (с альфа-каналом)

Конвертирует `.mov` видео с прозрачностью (альфа-канал) в `.webm` формат, сохраняя прозрачность.  
Использует кодек **VP9** — единственный кодек WebM, поддерживающий альфа-канал.

---

## Требования

- **Python 3.11+** (встроенные библиотеки: `tkinter`, `subprocess`, `threading`, `logging`)
- **ffmpeg** — должен быть установлен и доступен в `PATH`

### Установка ffmpeg

| Платформа | Команда |
|-----------|---------|
| Windows   | Скачать с https://ffmpeg.org/download.html → распаковать → добавить папку `bin` в PATH |
| macOS     | `brew install ffmpeg` |
| Linux     | `sudo apt install ffmpeg` или `sudo dnf install ffmpeg` |

---

## Запуск

```bash
python converter.py
```

---

## Использование

1. Нажмите **«Обзор...»** рядом с «Входной .mov файл» и выберите свой `.mov` файл
2. Укажите **путь к выходному файлу** (или нажмите «Обзор...» для выбора папки)
3. При желании настройте **качество (CRF)**:
   - `0` — максимальное качество, очень большой файл
   - `20` — хорошее качество (по умолчанию, рекомендуется)
   - `40+` — низкое качество, маленький файл
4. Нажмите **«▶ Начать конвертацию»**
5. Наблюдайте за прогрессом в процентах
6. Когда появится «✅ Видео готово!» — конвертация завершена

## Логи

Лог-файлы создаются автоматически в папке `logs/` рядом со скриптом.  
Имя файла: `conversion_YYYY-MM-DD_HH-MM-SS.log`

Логи содержат:
- Путь к входному и выходному файлу
- Размер файла
- Информацию о пиксельном формате (наличие альфа-канала)
- Длительность видео
- Точную команду ffmpeg
- Все ошибки ffmpeg (stderr)

## Технические детали

Используемая команда ffmpeg:
```
ffmpeg -y -i input.mov \
  -c:v libvpx-vp9 \
  -pix_fmt yuva420p \
  -auto-alt-ref 0 \
  -b:v 0 \
  -crf 20 \
  -row-mt 1 \
  -an \
  output.webm
```

- `-pix_fmt yuva420p` — формат с альфа-каналом (YUVA = Y, U, V + Alpha)
- `-auto-alt-ref 0` — обязательно при использовании альфа-канала
- `-b:v 0 -crf` — режим постоянного качества (VBR без ограничения битрейта)
- `-row-mt 1` — многопоточность по строкам (ускорение на многоядерных CPU)
- `-an` — без аудио (в .mov с прозрачностью аудио обычно нет)
